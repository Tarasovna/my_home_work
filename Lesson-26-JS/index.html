Syntax

console.log('Hello JavaScript!'); -- output information to the console

let

Creat new local variables. Это область памяти на которой мы будем храннить данные(число, строка)

const

Объявляет именованную константу, доступную только для чтения. то есть при изменеие после данных, т.е. присвоение нового значение, const - не даст ей измениться.

var

Объявляет переменную, инициализация переменной значением является необязательной.

Функции

Спец-й тип объкта, к-й позволяет формировать средствами языка определять логику поведения и обработки д-х.

alter

Нужна для взаиможействия с пользователем можно работать как с перемеными или математическими действиями.

promt

Предназначен для вывода диалогового окна с сообщением, текстовым полем для ввода данных и кнопками «ОК» и «Отмена». Это окно предназначено для запроса данных, которые пользователю нужно ввести в текстовое поле.

Рез-т promt можно разместить в переменную

Операторы

Могут быть как математические или остаток от деления.
